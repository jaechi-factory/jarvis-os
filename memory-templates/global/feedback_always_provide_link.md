변경 완료 시 항상 확인할 수 있는 링크를 함께 제공할 것.

**Why:** 사용자가 매번 별도로 요청하지 않아도 바로 확인할 수 있도록.
**How to apply:** 코드 수정 완료 보고 시 로컬 dev server URL(http://localhost:5173 등)을 항상 포함. 배포 후에는 프로덕션 URL도 함께 제공.
