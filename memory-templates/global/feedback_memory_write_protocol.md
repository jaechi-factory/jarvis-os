사용자가 피드백·규칙·결정사항을 **메모리에 박아달라고 요청하면 당 턴에 즉시 파일 작성 완료**한다. "다음에 반영"이나 "기억해두겠다"는 금지.

**Why:** 즉시 실행 안 하면 세션 압축/종료 시 규칙이 소실됨. 사용자가 반복 요청해야 하는 부담 발생. 2026-04-13 사용자 명시 요청.

**How to apply:**
- 트리거: "박아줘", "기억해", "저장해", "메모리에 넣어", "규칙으로 만들어", "영구 반영"
- 즉시 액션: 적절한 `memory/global/feedback_*.md` 또는 `memory/projects/project_*.md` 파일 Write → `MEMORY.md` 인덱스 업데이트
- 타입 분류: 작업 방식/선호 → feedback, 프로젝트 상태/결정 → project, 외부 시스템 경로 → reference, 사용자 프로필 → user
- 가드: 기존 메모리와 충돌하거나, 민감 정보 포함, 의도 불명확한 경우 **확인 먼저**. 그 외 즉시 실행
- 완료 보고: 어느 파일에 박았는지 경로 1줄로 알림
